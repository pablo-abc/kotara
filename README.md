# Kotara

A set of custom elements that help you make presentations on the browser.

## Usage

A presentation might look like this

```html
<html>
  <head>
    <script type="module">
      import 'kotara';
    </script>
  </head>
  <body>
    <kot-presentation>
      <kot-slide>
        <h1>This is the first slide</h1>
      </kot-slide>

      <kot-slide>
        <h1>This is the second slide</h1>
      </kot-slide>
    </kot-presentation>
  </body>
</html>
```
